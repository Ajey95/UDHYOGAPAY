\textbf{Note on Key Constraints:} All tables implement referential integrity using MongoDB ObjectId references with Schema-level validation. The relationships are enforced through:
\begin{itemize}
\item \texttt{ref} attribute in Schema definitions for foreign key relationships
\item \texttt{populate()} methods for JOIN-like operations
\item Application-level cascading operations for data consistency
\item Compound indexes on foreign key fields for optimized queries
\item Unique constraints on critical fields (email, phone, userId in workers)
\end{itemize}
For example, deleting a user triggers checks on related bookings and worker profiles, ensuring referential integrity through application-level transaction handling.

\textbullet~Coordinates must be valid [longitude, latitude] \newline
